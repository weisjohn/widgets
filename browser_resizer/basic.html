Browser Resizer

This helps test when Media Queries trigger on and off so that I don't have to constantly manually resize the browser as I'm working on Reponsive Web Designs.

Features:

  • Resizes the browser from maximum to minimum width and back again.
  • The resizing takes place over a set amount of time so that you can "see" the design respond.

How to install (desktop):

  • Drag this QR link to your bookmarks bar.
  • You now have a bookmarklet! Yay! You can now click the bookmarklet to generate a QR Code for any page that you're on.

How to install (mobile):

  • Click on this link (the page won't change in your browser)
  • Add a bookmark for this page.
  • Edit the bookmark and delete everything up to and including the "#" symbol.
  • You now have a bookmarklet! Yay! You can now click the bookmarklet to share any page that you're on.
